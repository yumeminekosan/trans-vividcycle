# 安全规范

> Queer Relationship Atlas — 安全策略、密钥管理、合规

## 核心原则

### 最小公开原则
- 默认 private，不是 public

### 隐私控制

| 可见性级别 | 说明 |
|-----------|------|
| private | 仅自己可见 |
| mutual | 双方互相关注可见 |
| friends_only | 朋友可见 |
| public | 公开 |

## 必须实现

### Anti-Scraping
- rate limit
- graph depth limit
- crawler detect

### 隐私保护
- 不允许默认导入真实姓名
- 不允许自动关联现实身份
