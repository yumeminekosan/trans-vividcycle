# 项目规划

> Queer Relationship Atlas — 以关系为核心的身份图谱平台

## 项目目标

构建一个 **"以关系为核心，而不是以算法推荐为核心"** 的身份图谱平台。

用户可以：
- 创建自己的身份节点
- 编辑关系网络
- 可视化社群连接
- 通过关系认识更多人
- 自定义关系语义
- 管理关系可见性
- 以联邦化方式跨社区连接

## 核心理念

### 传统社交平台
```
用户 -> 内容流 -> 算法推荐
```

### 本项目
```
用户 -> 关系 -> 社群拓扑 -> 连接发现
```

### 重点
- 非中心化关系
- 非实名制
- 非算法驱动
- 强隐私
- 强关系语义
- 可编辑人格节点

## 核心功能（MVP）

### Phase 1 必须实现

| 模块 | 功能 |
|------|------|
| 用户系统 | 注册、登录、Persona 创建、多 Persona 支持 |
| 关系系统 | 添加/编辑/删除关系，含标签、描述、可见性、强度 |
| Graph 可视化 | 节点显示、edge 显示、zoom、drag、relation hover |
| Profile 系统 | 自我介绍、pronouns、tags、links、federation handle |

## 用户类型

### 核心用户
- queer 用户
- neurodivergent 用户
- 同人创作者
- 黑客/技术社群
- 小型自治社区

### 次级用户
- 艺术家
- 研究者
- 联邦宇宙用户

## 暂时不做

- timeline
- 私信
- AI 推荐
- 短视频
- 手机 App
- ActivityPub federation
