# 术语表

> 统一术语，防止混乱

## 核心术语

| 术语 | 定义 |
|------|------|
| Persona | 用户的可组合身份节点 |
| Relation | 两个 persona 之间的关系边 |
| Community Node | 群体节点（乐队、黑客组织、zine 小组等） |
| Graph | 关系图谱的可视化 |
| Edge | 关系连接（同 Relation） |
| Visibility | 可见性控制级别 |

## 可见性级别

| 级别 | 说明 |
|------|------|
| private | 仅自己可见 |
| mutual | 双方互相关注可见 |
| friends_only | 朋友可见 |
| public | 公开 |

---

_文件职责：统一术语，防止 edge/relation/connection/link 混乱_
