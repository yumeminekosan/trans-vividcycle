# 愿景

> 这个项目最终想成为什么

## 长期愿景

构建一个 **"以关系为核心，而不是以算法推荐为核心"** 的身份图谱平台。

## 项目气质

- 非中心化关系
- 非实名制
- 非算法驱动
- 强隐私
- 强关系语义
- 可编辑人格节点

## 世界观

### 传统社交平台
```
用户 -> 内容流 -> 算法推荐
```

### 本项目
```
用户 -> 关系 -> 社群拓扑 -> 连接发现
```

## 社群目标

为 queer、neurodivergent、同人创作者、黑客/技术社群、小型自治社区提供一个以关系为纽带的身份图谱空间。
