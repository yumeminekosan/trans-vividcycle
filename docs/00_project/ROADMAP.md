# 路线图

> 时间阶段

## Stage 0 — 概念验证（1~2 周）

- graph 能显示
- 用户能创建 persona
- 用户能连线

## Stage 1 — MVP（1~2 月）

- 基础用户系统
- graph 编辑
- relation visibility

## Stage 2 — 社群化（2 月）

- recommendation
- shared communities
- graph discovery

## Stage 3 — 联邦化（2~4 月）

- ActivityPub
- Mastodon import
- remote relation

## Stage 4 — 实时协作（长期）

- CRDT
- 多人编辑 graph
- live relation editing
