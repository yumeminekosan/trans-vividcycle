# 用户流程

> 用户路径

```
注册
  ↓
创建 Persona
  ↓
添加关系
  ↓
浏览 graph
  ↓
发现社群
```

## 详细步骤

### 1. 注册
- 输入用户名
- 设置密码
- 可选：邮箱

### 2. 创建 Persona
- 选择名称
- 填写 bio
- 设置 pronouns
- 添加 tags

### 3. 添加关系
- 搜索 persona
- 选择关系类型
- 设置可见性
- 可选：描述、权重

### 4. 浏览 Graph
- 缩放
- 拖拽
- 点击节点查看详情
- 点击 edge 查看关系
